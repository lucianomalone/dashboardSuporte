# 📊 Relatório de Incidência de Chamados

Dashboard interativo desenvolvido em **Python + Streamlit**, com o objetivo de analisar chamados de suporte técnico, identificando os serviços e categorias com maior incidência.

🔗 Aplicação publicada via **Streamlit Cloud**.

---

## 🎯 Objetivo
Fornecer uma visão clara e executiva sobre:
- Serviços que mais geram chamados
- Categorias mais recorrentes
- Combinações críticas de Serviço + Categoria

Auxiliando na tomada de decisão para:
- Criação de FAQ
- Treinamento de usuários
- Melhoria de processos
- Prioridade de correções

---

## 🧠 Funcionalidades
- 📂 Upload de arquivo Excel ou CSV
- 📊 KPIs automáticos
- 🥇 Top 3 Serviços com mais chamados
- 🥈 Top 3 Categorias com mais chamados
- 🥉 Top 3 combinações de Serviço + Categoria
- 🧠 Insights automáticos
- 📈 Gráficos interativos (Plotly)
